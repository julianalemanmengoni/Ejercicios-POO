# Ejercicio 18 - T-Rex Extremo

Juego basado en el T-Rex de Google Chrome. Implementado en Qt usando colores simples (sin imágenes externas).

## Características

- Dinosaurio puede saltar, agacharse, avanzar y frenar.
- Obstáculos: cactus (QTimer principal) y pájaros (cada uno con su QTimer).
- Aumenta la dificultad con el tiempo.
- Colisiones terminan el juego.
- Presionar Espacio para reiniciar.

## Controles

- Espacio: Saltar
- Flecha Abajo: Agacharse
- Flecha Derecha: Adelantarse
- Flecha Izquierda: Frenar

## Cómo correr

1. Abrí `ejercicio18.pro` con Qt Creator.
2. Compilá y ejecutá.
